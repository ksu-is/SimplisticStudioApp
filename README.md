### Overview
The **Simplistic Studio App** is a digital workspace built for small clothing brands and independent fashion designers to bring their creative ideas to life.  
It allows users to upload apparel designs, generate product mockups, track inventory, and connect to marketing tools such as Instagram or TikTok for automated drop campaigns.  

By combining **creativity** with **business analytics**, Simplistic Studio helps entrepreneurs run their clothing brand efficiently â€” all from one dashboard.

---

### Key Features
- Upload and preview clothing designs (hoodies, shirts, hats, etc.)
- Generate realistic mockups using Python image processing
- Manage inventory and product data with a built-in database
- Track engagement and sales trends through analytics charts
- Integrate with social media APIs for automated post scheduling
- Connect with suppliers and manage production updates

---

### Technology Stack
- **Backend:** Python (Flask framework)
- **Frontend:** React (for interactive UI)
- **Database:** SQLite / PostgreS
